
🧠 Notions à Apprendre / Maîtriser

---

🧵 Gestion des processus

- fork() : Création d’un processus fils pour exécuter une commande.
- execve() : Exécution d’un binaire, en remplaçant le processus courant.
- wait(), waitpid() : Attente de la fin d’un processus fils.
- pipe() : Création d’un canal de communication entre deux processus.
- Signal Handling : signal(), sigaction() pour intercepter SIGINT, SIGQUIT, etc.

---

📂 Gestion des fichiers et redirections

- Fonctions basiques : open(), read(), write(), close()
- Duplication de descripteurs : dup(), dup2() (pour rediriger stdin/stdout)
- Redirections à implémenter :
  - < : redirection d’entrée
  - > : redirection de sortie
  - >> : redirection de sortie en mode ajout
  - << : heredoc, lire jusqu'à un délimiteur

---

🐚 Parsing (analyse de ligne de commande)

- Séparer les commandes par |
- Identifier et stocker les redirections
- Gérer correctement les quotes ' et " :
  - '...' : empêche toute expansion
  - "..." : autorise l'expansion de $
- Expansion de :
  - Variables d’environnement : $VAR
  - Code de sortie de la dernière commande : $?

---

🔀 Built-in Commands

- echo (avec -n)
- cd (chemins relatifs ou absolus)
- pwd (aucune option)
- env (sans argument)
- export, unset
- exit (aucune option)
- Manipuler l’environnement :
  - getenv(), setenv(), unsetenv(), variable environ

---

📜 Terminal & readline

- Fonctions : readline(), add_history(), rl_clear_history()
- Affichage d’un prompt minimaliste
- Gestion des signaux et du comportement clavier :
  - ctrl-C : nouvelle ligne + nouveau prompt
  - ctrl-D : quitte le shell
  - ctrl-\ : ne fait rien (en mode interactif)

---

🧠 Structures de données

- Manipuler des tableaux de chaînes : char **env
- Structures personnalisées pour :
  - Stocker une commande
  - Représenter des redirections
  - Gérer des listes de tokens ou des pipelines

---

🧼 Gestion de la mémoire

- Zéro memory leaks :
  - Toujours free() ce qui est alloué
  - Utilisation de valgrind obligatoire
- Ne pas oublier les erreurs fréquentes :
  - Double free
  - Non-libération après une erreur
  - Allocation partielle non nettoyée
